Africadre dev env
=================

Requirements
----
- vagrant + shell + virtualbox
- ubuntu/xenial64
- php7
- nginx
- mysql 5.7

How to install ?
--------------
In the host machine, ensure vagrant and virtualbox are installed

```shell
$ git clone git@github.com:abarry/africadre-dev-env.git africadre-dev-env
$ cd africadre-dev-env
$ vagrant up --provision
```
